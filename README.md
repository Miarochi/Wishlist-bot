# Wishlist Bot

Telegram-бот для одного владельца: друзья сами заполняют анкету о себе (дата рождения и пожелания к подаркам),
а бот присылает владельцу напоминание за N дней до дня рождения друга.

## Как это работает

1. Владелец делится ссылкой на бота (`https://t.me/<имя_бота>`) с друзьями — приглашать каждого по отдельности не нужно.
2. Друг отправляет боту `/start`. Имя бот берёт из профиля Telegram автоматически и спрашивает его только если
   оно пустое.
3. Бот спрашивает дату рождения в формате `ДД.ММ.ГГГГ` (Telegram не передаёт эти данные ботам, так что этот
   вопрос — всегда). Если формат не совпадает, бот просит прислать дату ещё раз в нужном виде.
4. Бот просит прислать ссылку(и) на вишлист (Wildberries/Ozon/Kaspi) или просто описать словами, что друг хочет
   получить в подарок — можно несколько сообщений подряд, в конце друг нажимает «Готово».
5. Владелец сразу получает уведомление со всеми данными анкеты.
6. За `REMINDER_DAYS_BEFORE` дней до дня рождения бот присылает владельцу дату, пожелания и заметки, с кнопками
   «Спросить у друга об обновлении» / «Не сейчас».

**Ограничение**: бот видит только сообщения, которые друзья пишут ему напрямую — не личную переписку владельца с ними.
Ссылки и текстовые пожелания хранятся как есть, без автоматического парсинга или подбора похожих товаров.

## Установка

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
```

Заполните `.env`:
- `BOT_TOKEN` — получить у [@BotFather](https://t.me/BotFather)
- `OWNER_ID` — ваш Telegram user id, узнать у [@userinfobot](https://t.me/userinfobot)

## Запуск

```bash
python -m bot.main
```

## Меню владельца

Владелец управляет ботом через кнопки внизу экрана (reply-клавиатура) — они прикреплены к каждому ответу бота,
поэтому не пропадают:

- 📋 **Статус анкет** — все друзья и статус их анкеты
- 👤 **Подробности** — выбор друга из списка кнопками → пожелания, заметки, статус анкеты
- 🔄 **Попросить** — выбор друга кнопками → бот просит его прислать вишлист заново
- ✏️ **Редактировать** — редактирование анкеты целиком в одном сообщении: выбираешь друга → бот показывает
  карточку с кнопками «Имя / Дата рождения / Вишлист / Заметки» → тап по полю превращает то же сообщение
  в запрос нового значения → после ответа сообщение возвращается в обновлённую карточку. Значение поля
  **заменяется**. Кнопка заметок сама подстраивается под ситуацию: «📝 Добавить заметку», если их ещё нет,
  и «📝 Заметки» (с заменой текста), если уже есть.

Слэш-команды (`/status`, `/friend`, `/ask`, `/edit`, `/help`) доступны через системное меню Telegram и дублируют
кнопки — если набрать их без имени, бот тоже покажет выбор друга кнопками.

Ещё пара команд есть только в меню Telegram, отдельных кнопок под них нет:

- **`/delete`** — удалить друга вместе со всеми данными, с обязательным подтверждением («🗑 Да, удалить» /
  «❌ Отмена») — действие необратимо.
- **`/backup`** — присылает файл `wishlist_bot.db` с текущей базой данных.

Друга всегда выбираешь тапом по кнопке со списком имён — вводить имя вручную не нужно. Если у двух друзей
совпадает имя из Telegram, они будут отдельными пунктами в списке выбора, различить их можно по нику (@…) в
карточке.
